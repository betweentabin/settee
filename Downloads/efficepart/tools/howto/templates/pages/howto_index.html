目次生成機能の使い方

機能概要

マニュアルやドキュメントの目次を自動で生成する機能です。Word/PDFファイルをアップロードするだけで、見やすい目次を作成できます。

基本的な使い方

  • 対象ファイルをアップロード
  • 目次の形式を選択
  • 生成ボタンをクリック
  • 生成された目次をダウンロード

対応ファイル形式

アップロード : PowerPoint(.pptx)

ダウンロード : PowerPoint(.pptx)

文章校正機能の使い方

機能概要

文章の校正を自動で行い、より良い表現を提案する機能です。文法チェックや表現の改善など、プロフェッショナルな文章作成をサポートします。

基本的な使い方

ファイル校正

  • 校正したいファイルをアップロード
  • 校正スタイルを選択（ビジネス文書・論文など）
  • 校正を実行
  • 提案された改善点を確認・適用

テキスト校正

  • 校正スタイルを選択（ビジネス文書・論文など）
  • 校正したい文章を入力欄に入力
  • 校正を実行
  • 提案された改善点を確認・適用

対応ファイル形式

アップロード : PowerPoint(.pptx)、Word(.docx)、PDF(.pdf)、テキストファイル(.txt)

ダウンロード : PowerPoint(.pptx)、Word(.docx)、PDF(.pdf)、テキストファイル(.txt)

休憩シフト機能の使い方

機能概要

従業員の休憩スケジュールを最適化する機能です。勤務時間や希望条件を考慮しながら、効率的な休憩シフトを自動で作成します。

基本的な使い方

  • スタッフ人数を入力
  • ポジション名を入力
  • 臨時ポジション名を入力
  • 勤務時間帯を入力
  • 休憩時間を入力
  • 休憩時間帯を入力
  • 個別休憩時間の有無を選択
  • あり：個別休憩時間、何週目の休憩かを入力
  • 休憩シフト生成を実行

対応ファイル形式

アップロード : なし

ダウンロード : Excel(.xlsx)、PDF(.pdf)

名札生成機能の使い方

機能概要

イベントや会議用の名札を一括生成する機能です。参加者リストから、美しいデザインの名札を効率的に作成できます。

基本的な使い方

  • テンプレートのExcelファイルをダウンロード
  • 名札情報を入力したテンプレートのExcelファイルをアップロード
  • デザインをカスタマイズ（サイズ、背景、フォント、テキスト位置）
  • トリムマーク、切断線の有無を選択
  • ダウンロード形式を選択
  • ダウンロードを実行

対応ファイル形式

アップロード : Excel(.xlsx)

ダウンロード : PDF(.pdf)、JPEG(.jpg)、SVG(.svg)

形式変換機能の使い方

機能概要

様々なファイル形式を相互に変換する機能です。ドキュメント、画像、動画など、多様なファイル形式に対応しています。

基本的な使い方

  • 変換したいファイルをアップロード
  • 出力形式を指定
  • 解像度を設定
  • 変換を実行

対応ファイル形式

アップロード : JPEG(.jpg)、PNG(.png)、PDF(.pdf)、SVG(.svg)

ダウンロード : JPEG(.jpg)、PNG(.png)、PDF(.pdf)、SVG(.svg)

圧縮転送機能の使い方

機能概要

ファイルを圧縮して効率的に共有する機能です。大容量ファイルも手軽に送信できます。

基本的な使い方

  • ファイルを選択またはドラッグ＆ドロップ
  • 圧縮レベルを選択
  • 共有方法を選択
  • URLまたはQRコードで共有

対応ファイル形式

アップロード : すべてのファイル形式

ダウンロード : すべてのファイル形式

PDFツール機能の使い方

機能概要

PDFファイルを編集・加工する機能です。ページの結合、分割、抽出、削除などが可能です。

基本的な使い方

PDF結合

  • 結合したい複数のPDFファイルをアップロード
  • 結合を実行

PDF分割

  • 分割したいPDFファイルをアップロード
  • 分割したい箇所の分割線を選択
  • 分割を実行

PDF抽出

  • 抽出したいPDFファイルをアップロード
  • 抽出したいページを選択
  • 抽出を実行

PDF削除

  • 削除したいPDFファイルをアップロード
  • 削除したいページを選択
  • 削除を実行

対応ファイル形式

アップロード : PDF(.pdf)

ダウンロード : PDF(.pdf)
